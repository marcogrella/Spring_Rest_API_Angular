Olá! Bem vindo!

Este um projeto de executado com as seguintes tecnologias. No Back-End foi utilizado Java e SpringBoot. O sistema possui validação por Token e senha com criptografia. Já no Front-End foi utilizado Angular com Bootstrap 5.

Pontos Positivos:

* Validação de Campos;

* Paginação de registros;

* Consulta;

* Segurança utilizando Token;

* Gráficos em tempo real;
